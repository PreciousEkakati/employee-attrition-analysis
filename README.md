# Employee Attrition Analysis

## Project Overview

This project investigates employee attrition trends within a software provider company to identify factors contributing to turnover and recommend retention strategies.

## Business Problem

The organization experienced rising employee attrition and required data-driven insights to improve retention.

## Dataset

- Employee demographics
- Department information
- Job roles
- Satisfaction scores
- Attrition status

## Tools Used

- Excel
- Power Query
- Power BI

## Dashboard Features

- Attrition Rate Tracking
- Department-Level Analysis
- Employee Satisfaction Analysis
- Demographic Insights
- Interactive Filtering

## Dashboard Preview

![Dashboard](https://github.com/PreciousEkakati/employee-attrition-analysis/blob/main/Employee%20Attrition%20Analysis.jpg?raw=true)

## Key Insights

- Certain departments recorded higher attrition rates.
- Lower satisfaction scores correlated with increased turnover.
- Specific employee groups were more likely to leave.

## Recommendations

- Strengthen engagement initiatives in high-risk departments.
- Improve employee satisfaction monitoring.
- Implement targeted retention programs.

## Business Impact

Insights contributed to retention strategies that supported a reduction in monthly attrition rates.
